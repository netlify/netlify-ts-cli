# SaaS Example

A SaaS landing page built with TanStack Start and Tailwind CSS for Netlify deployment.

## Features

- **Hero Section**: Bold headline with call-to-action buttons
- **Feature Grid**: Four feature cards showcasing product capabilities
- **Pricing Tiers**: Three-tier pricing (Starter, Pro, Enterprise)
- **FAQ Page**: Accordion-style frequently asked questions

## Development

```bash
npm run dev
```
